20) Carrega l’exercici 000.json de la carpeta ‘./public/exercicisCSS/’ a la eina.

En CSS les pseudo-classes permeten definir estils segons en quina situació està un element HTML. Les més importants són:

:hover quan el ratolí està per sobre d’un element

:visited en els enllaços <a> quan aquell enllaç ja s’ha visitat alguna vegada

:first-child el primer element d’un tipus en un grup (com el primer <td> en una taula)

:last-child l’ultim element d’un tipus en un grup (com l’últim <li> en una llista)

:nth-child() element número (X) d’un grup

:focus quan un element té el ‘focus’, per exemple en un formulari quan està seleccionat

:disabled quan un element té l’atribut ‘disabled’ (i està desactivat)

La pseudo classe :hover permet modificar l’estil d’un element quan el ratolí passa per sobre. A l’exemple /exercicisCSS/020.css pots veure com es fan servir els pseudo-elements:

.quadre0 {

   background-color: lightskyblue;

   height: 50px;

   width: 100px;

}

.quadre0:hover {

   background-color: black;

   width: 200px;

}

En l’exemple anterior, quan l’element amb class ‘quadre0’ tingui el ratolí pel damunt, la seva mida serà de 200 píxels enlloc de 100. I a més, serà de color negre:

Important: Fixa’t que la propietat ‘hover’ no té en compte els marges, però si els borders i els paddings. Modifica el ‘quadre1’ perquè tingui el següent comportament:

  • Radi del border 50px

En CSS les transicions permeten que el navegador faci una animació entre dos valors diferents quan hi ha un canvi entre els valors (per exemple quan un s’aplica una pseudo-clase o un codi javascript canvia el valor d’una propietat).

  • Les transicions només funcionen per propietats numèriques (i colors)
  • Les transicions s’activen automàticament quan canvia un valor
  • Es poden definir transicions a múltiples valors, separant-les per comes

Les transicions tenen els següents paràmetres:

  • Propietat a la que s’aplica la transició
  • Duració de la transició (s significa en segons, ms en milisegons)
  • Funció a aplicar (podeu veure exemples de cada funció aquí)
  • Retard (temps que triga) a aplicar la transició, aquest paràmetre és opcional

En el nostre exercici:

transition: width 0.5s ease, background-color 1s linear 0.5s;

  • Primer fa una transició de ample que dura mig segon
  • Al cap de mig segon comença una transició de color que dura 1 segon

Modifica l’exercici 1 perquè el segon quadre es comporti així:

25) Carrega l’exercici 025.json de la carpeta ‘.public/exercicisCSS/’ a la eina.

A part de les transicions, que permeten fer una animació entre canvis de valors d’una propietat. CSS permet definir animacions que permeten més complexitat i valors intermitjos. Per definir una animació fan falta dues parts:

@keyframes: defineixen el valor de les propietats a cada punt de l’animació, poden definir diferents propietats al mateix temps. Dos exemples:

@keyframes example {

   from        { background-color: red;                 top: 0;     left: 0; }

   to                { background-color: yellow;         top: 50px;  left: 50px; }

}

@keyframes framesQuadre {

   0%      { background-color: lightskyblue;   top: 0;     left: 0; }

   33%     { background-color: lightsalmon;    top: 50px;  left: 50px; }

   66%     { background-color: lightgreen;     top: 0px;   left: 100px; }

   100%    { background-color: lightskyblue;   top: 0;     left: 0; }

}

Fixa’t que es pot posar simplement “from” i “to” per definir l’estat de cada propietat al principi i al final de l’animació. O també es pot posar l’estat en cada percentatge de l’animació, on 0% equival a “from” i 100% equival a “to”.

Un cop definits els estats clau de l’animació, cal dir a la classe que faci servir aquestes animacions, això es fa amb la propietat ‘animation-name’ com en el següent exemples:

.quadre {

   animation-name: framesQuadre;

   animation-duration: 4s;

   background-color: lightskyblue;

   left: 0;

   height: 100px;

   position: absolute;

   top: 0;

   width: 100px;

}

Altres propietats que es poden definir:

animation-name                : nom dels keyframes que ha de fer servir

animation-duration                : duració de l’animació

animation-delay                : retard a començar l’animació

animation-iteration-count        : cops que es repeteix una animació

animation-direction                : si la animació va del 0 al 100, del 100 al 0 o alternant

animation-timing-function        : funció de timing (igual que a les transicions; linear, ease, …)

animation-fill-mode                : com s’apliquen les propietats quan l’animació està “quieta”
